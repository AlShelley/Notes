JavaScript基础

2019/1/24 12:35 下午 posted in  开发

JavaScript是一种脚本编程语言。 JavaScript运行顺序是从上往下，因此要注意代码的顺序。

如何使用JavaScript

内部JavaScript：在html页面中嵌入<script>这里写js代码</script>即可插入js代码。 外部JavaScript：<script src="script.js"></script>可以调用html文件同级目录下的script.js文件。

注释

// 这是单行注释 /* 这是多行注释 */

变量

var用来定义变量

函数

function funName(){ js代码 } 函数是可复用的代码块，可以一次编写反复使用。

运算符

+、-、*、/ 分别是加减乘除 ===、!==、<、> 分别是等于、不等于、小于、大于 &&、||、! 分别表示与、或、非

条件语句

if(条件){ js代码 } else { js代码 }

事件

事件相关函数分为：事件监听器和事件处理器

循环

for(起式值；退出条件；增加器){ js代码 }

switch语句

switch - 他们以单个表达式/值作为输入，然后查看多个选项，直到找到与该值相匹配的选项，执行与之相关的代码。

switch (expression) {
  case choice1:
    run this code
    break;

  case choice2:
    run this code instead
    break;
    
  // include as many cases as you like

  default:
    actually, just run this code
}
