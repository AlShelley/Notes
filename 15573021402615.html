Mac 上的 svnGUI 工具：smartsvn 安装、激活和使用

2019/5/8 15:55 下午 发表于  工具

smartsvn 是 mac 上的 svn 客户端，svn 是类似 git 的版本控制工具。

1、下载 smartsvn srmatsvn 官网下载地址
2、在任意目录下新建文件 smartsvn.license，然后写入以下内容：

Name=csdn  
Address=1337 iNViSiBLE Str.  
Email=admin@csdn.net  
FreeUpdatesUntil=2099-09-26  
LicenseCount=1337  
Addon-xMerge=true  
Addon-API=true  
Enterprise=true  
Key=4kl-<Zqcm-iUF7I-IVmYG-XAyvv-KYRoC-xlgsv-sSBds-VAnP6

3、在安装软件需要导入 license 时选择上述文件或者安装软件后在菜单栏的 help-register 中导入密钥文件即可激活软件。

4、点击菜单栏的project-checkout，添加远程目录，然后输入认证信息，输入本地目录信息，即可挂载远程目录到本地。

5、若认证出错，则点击菜单栏的 SmartSVN-preferences，然后选择 Authentication，使用 delete 删除之前存储的认证信息，然后重新 checkout 即可。

Mac 终端下 svn 常用命令

Mac 可以不用安装 svnGUI 而直接使用命令进行 svn 操作，常见svn 命令如下：
svn help 查看命令帮助
svn checkout xxx 拉取远程目录到本地
svn add file 往版本库中添加新文件

更多常用命令可参考：Mac终端下svn常用命令

通过命令行操作进行认证后，其认证信息存储在 ~/.subversion/auth/svn.simple目录下，若认证失败，需要在此目录下情况认证信息后重新进行认证。
