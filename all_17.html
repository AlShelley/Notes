使用python快速搭建ftp服务器

使用第三方库：pyftpdlib
安装库：pip3 install pyftpdlib
其他依赖项：
pip3 install PyOpenSSL 用来支持ftps。
pip3 install pysendfile 用来在unix上加速上传（服务器到客户端）。
安装python库的时候要注意大小写。

后面的事情就是看官方示例，然后copy了。

使用ssh科学上网

打开终端输入：
ssh -D 1085 -p 6754 root@xx.xx.xx.xx
然后输入密码即可。xx即远程服务器ip。
这意味着本地1085端口被转发到了远程服务器的6754上，然后在浏览器配置socks5代理：127.0.0.1 1085
就可以科学上网了。

如果不稳定，可以考虑使用autossh。

修改Mac主机名、计算机名

$sudo scutil --set ComputerName 你想要的计算机名称
$sudo scutil --set HostName 你想要的主机名称

举例：
$sudo scutil --set HostName lonelyor

使用以下命令查看主机名或计算机名
$ scutil --get ComputerName
$ scutil --get HostName
