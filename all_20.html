fluxgui安装与卸载

很久以前的笔记了

Ubuntu/Debian Manual Install
# Install dependencies
sudo apt-get install git python-appindicator python-xdg python-pexpect python-gconf python-gtk2 python-glade2 libxxf86vm1

# Download fluxgui
cd /tmp
git clone "https://github.com/xflux-gui/fluxgui.git"
cd fluxgui
python download-xflux.py

# EITHER install globally
sudo python setup.py install
# EXCLUSIVE OR, install in your home directory. The binary installs
# into ~/.local/bin, so be sure to add that to your PATH if installing locally.
python setup.py install --user

# Run flux
fluxgui
Ubuntu/Debian Manual Uninstall
If you manually installed instead of using the PPA, you can uninstall by making setup.py tell you where it installed files and then removing the installed files.

# EITHER uninstall globally
sudo python setup.py install --record installed.txt
sudo xargs rm -vr < installed.txt

# EXCLUSIVE OR uinstall in your home directory
python setup.py install --user --record installed.txt
xargs rm -vr < installed.txt

backup系统备份工具介绍

backup是一款强大的备份软件，用于系统备份、增量备份、远程备份，企业级。 我还是当年使用debian时用过...这是以前的笔记，留一份在这好了。

软件安装（debian 9） apt-get install backuppc 然后会弹窗： backuppc设定 管理web地址： http://lonelyor/backuppc user：backuppc passwd：u6oN2KE1 可以通过运行htpasswd /etc/backuppc/htpasswd来修改密码 这些信息要记住！

安装后： Creating config file /etc/backuppc/config.pl with new version（可以暂时忽略）

然后运行： systemctl restart apache2 /etc/init.d/backuppc restart

接着就打开浏览器访问： http://lonelyor/backuppc 就能看到web管理界面了。 如果进不去可以打开隐身模式或者清除cookie再试。

官方安装文档：http://backuppc.sourceforge.net/BackupPC-4.1.3.html#Installing-BackupPC

参考文档： http://server.zol.com.cn/513/5130348.html

使用ssh科学上网

打开终端输入： ssh -D 1085 -p 6754 root@xx.xx.xx.xx 然后输入密码即可。xx即远程服务器ip。 这意味着本地1085端口被转发到了远程服务器的6754上，然后在浏览器配置socks5代理：127.0.0.1 1085 就可以科学上网了。

如果不稳定，可以考虑使用autossh。

扫描网站备份文件思路

sk15大佬提供一个思路，使用备份文件名list+域名list2扫描，这样每次循环都是一个备份文件名扫描list2次。 如果使用域名list2+备份文件名list1则每次循环都是一个域名就会扫描list1次（这样由于请求频次高容易导致服务器宕机）

大佬还提供了一个更有意思的方法，将list1和list2做成一个list，然后将其打乱，再扫描。（骚思路就像好算法，是编程的精髓。）
