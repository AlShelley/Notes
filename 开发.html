c 语言中变量前面加`*`表示什么？如`*`p

*p的意思是：p是一个指针（即指向某一个区域的地址）
如果前面加了修饰，就表示为那个类型的指针
如：int *p;表示p是一个整型指针，指向一个整型变量的地址

python 计算代码运行时间的方法

话不多说，直接上代码：
方法一：

import time
start = time.time()
run_function()
end = time.time()
print(end)

方法二：

import time
start = time.clock()
run_function()
end = time.clock()
print(end-start)

秒级计算

上述方法 print 的结果是原始时间数据，可以对其进行加减运算，但是结果不太直观，通常我们统计时间是以秒为单位。
下面给出秒级计算的代码:

import time
start = time.time()
run_function()
end = time.time()
print(int(end))

扩展

import time
import datetime

t = time.time()

print (t)                       #原始时间数据
print (int(t))                  #秒级时间戳
print (int(round(t * 1000)))    #毫秒级时间戳

nowTime = lambda:int(round(t * 1000))
print (nowTime());              #毫秒级时间戳，基于lambda

print (datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S'))   #日期格式化

1554358917.381631
1554358917

将秒级时间戳转为日期 

ts = 1515774430
dt = time.strftime("%Y-%m-%d %H:%M:%S", time.localtime(ts))
print(dt)

返回

python程序退出的几种方式

最常见的方式是使用 sys 库中的 sys.exit() 方法。不需要考虑平台等因素的影响，一般是退出Python程序的首选方法。

该方法中包含一个参数status，默认为0，表示正常退出，也可以为1，表示异常退出。

sys.exit()
sys.exit(0)
sys.exit(1)

该方法引发的是一个SystemExit异常(这是唯一一个不会被认为是错误的异常)，当没有设置捕获这个异常将会直接退出程序执行，当然也可以捕获这个异常进行一些其他操作。

其他方法非首选因此不详细介绍，这里简单提一下，具体信息自行百度。

其他方法有：
os._exit()
os.kill() 仅用于 unix 平台
os.popen() 仅用于 windows 平台

JavaScript基础

JavaScript是一种脚本编程语言。
JavaScript运行顺序是从上往下，因此要注意代码的顺序。

如何使用JavaScript

内部JavaScript：在html页面中嵌入<script>这里写js代码</script>即可插入js代码。
外部JavaScript：<script src="script.js"></script>可以调用html文件同级目录下的script.js文件。

注释

// 这是单行注释
/* 这是多行注释 */

变量

var用来定义变量

函数

function funName(){
js代码
}
函数是可复用的代码块，可以一次编写反复使用。

运算符

+、-、*、/ 分别是加减乘除
===、!==、<、> 分别是等于、不等于、小于、大于
&&、||、! 分别表示与、或、非

条件语句

if(条件){
js代码
} else {
js代码
}

事件

事件相关函数分为：事件监听器和事件处理器

循环

js代码
}

switch语句

switch - 他们以单个表达式/值作为输入，然后查看多个选项，直到找到与该值相匹配的选项，执行与之相关的代码。

switch (expression) {
  case choice1:
    run this code
    break;

  case choice2:
    run this code instead
    break;
    
  // include as many cases as you like

  default:
    actually, just run this code
}
